1. To Do List <br>
   Built an advanced desktop To-Do List application using C++ with a graphical user interface to manage daily tasks efficiently.<br>
   The application supports task creation, editing, deletion, priority levels, due dates, and real-time status updates.<br>
   Implemented local data storage to ensure tasks persist between sessions.<br>
   Focused on modular design, optimized performance, and a responsive GUI layout to enhance usability and scalability.<br>
